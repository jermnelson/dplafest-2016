Lua Scripting in Redis

The design of the Linked-Data Fragment Server Redis LRU Cache shamelessly copies Redis in representing triples in the cache. Much like how the SHA1 of a Lua Script is used to execute server-side code in Redis; in the LDFS Redis LRU Cache design all subjects, predicates, and objects are stored as string values using the SHA1 hash of the value as the Redis key.
