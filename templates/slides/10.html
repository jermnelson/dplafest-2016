Redis Cluster Implementation

With the recent release of Redis 3.0, Redis now has a cluster mode that allows you to treat multiple Redis instances as a single Redis server (with some limitations). This makes it possible to have datasets that exceed the memory of an single Redis instance.

KnowledgeLinks RDF Framework Integration

The RDFFramework is a new open-source project that allows you define your application in whatever preferred RDF vocabulary and then augmented with KnowledgeLinks RDF vocabulary to generate dynamic HTML CRUD forms and APIs.
