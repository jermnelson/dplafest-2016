What is Redis?

Redis, short for REmote DIctionary Server, is an in-memory key-value data storage server and is one of the most popular No-SQL technologies. Started and still actively maintained by Salvatore Sanfilippo, Redis use as sophisticated "glue" technology powers solutions across both start-ups and established technology companies such as Twitter, Slack, Medium, and Cisco.

Redis real power comes from the rich set of operations you can perform on different data structures like lists, sets, hashes, sorted sets, and other more exotic data types.
