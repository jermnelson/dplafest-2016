What is Redis?

Redis, short for REmote DIctionary Server, is an in-memory key-value data storage server and is one of the most popular No-SQL technologies. Started and still actively maintained by Salvatore Sanfilippo, Redis use as sophisticated "glue" technology powers solutions across both start-ups and established technology companies such as Twitter, Slack, Medium, and Cisco.

Redis real power comes from the rich set of operations you can perform on different data structures like lists, sets, hashes, sorted sets, and other more exotic data types.

Data Structure Description Used in LDFS?
String Redis most basic data structure type is a string value that Redis stores and manipulates as characters. Yes
List A collection of values that preserves order, values can be integers, strings, floats, and can be mixed and matched. Duplicate values are allowed No
Hash A collection that maps one or more fields to corresponding value pairs. In Redis, all hash values must be Redis strings, integers, or floats with unique field names. Yes*
Set A set is a collection of string values where uniqueness of each member is guaranteed but does not offer ordering of members. Yes*
Sorted Set A sorted set combines characteristics of both Redis lists and sets. Like a Redis list, a sorted set's values are ordered and like a set, each value is assured to be unique. No
